📄 API Reference
GET /students Returns all students
→ { success: true, count: 5, data: [ ... ] }
GET /students/:id Get student by ID
→ { success: true, data: { id, name, branch, year } }
POST /students Add new student
Body: { "name": "...", "branch": "...", "year": 2 } → { success: true, message: "Student added successfully", data: { ... } }
PATCH /students/:id Update student fields
Body: { "name"?: "...", "branch"?: "...", "year"?: 3 } → { success: true, message: "Student updated successfully", data: { ... } }
DELETE /students/:id Delete a student
→ { success: true, message: "Student deleted successfully", data: { ... } }
🔧 Postman Setup

1. Open Postman → New Collection → Student Records API
2. Set Base URL variable: http://localhost:3000
3. For POST/PATCH: Body → raw → JSON
4. Import the StudentRecords.postman_collection.json included in the ZIP
